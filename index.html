About the Project

The Assignment:

This project is an assignment in Choate Rosemary Hall's Machine Learning class (CS580-HO). Individually, we each were required to use Google Colab Notebooks to create a machine learning project.

I decided to use the DeOlidfy Algorithm to colorize old photos from the Choate Rosemary Hall archives.

About DeOldify:

While brainstorming new project ideas for this Machine Learning project, I came upon this project by jantic. He created a project called DeOlidfy that is can "colorize and restore old images and film footage." The Project's GitHub Page has some amazing examples of the capabilities of the project. If you would like to learn how the project works, please visit the GitHub page. Additionally, the project has been implemented on the website MyHeritage.
